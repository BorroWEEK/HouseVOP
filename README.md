# HouseVOP

Вёрстка макета для самостоятельной практики. Сайт жилого комплекса в историческом комплексе.

### В основе проекта лежат такие технологии как:
* flex-box;
* position;
* animation;
* transform;
* transition;
* Методология БЭМ nested;
* Псевдоклассы;
* Псевдоэлементы;
* Forms;
